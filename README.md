# ENCOM's Hits

Retro arcade game collection in Cyrius. Four games, one engine, neon on black.

## The Games

| Game | Mechanic | Lines (est.) |
|------|----------|-------------|
| **Light Cycles** | Trail walls, last alive wins | ~200 |
| **Grid Bugs** | Grid traversal, pathfinding pursuit | ~300 |
| **Battle Tanks** | Maze combat, ricochet projectiles | ~400 |
| **MCP Cone** | Rotating barriers, timing-based | ~300 |

## Build

```sh
cyrius build src/main.cyr build/encom-hits
./build/encom-hits
```

## Visual Style

Neon wireframe on black. No textures, no sprites, no assets. Every pixel is a computed line. The art is math.
